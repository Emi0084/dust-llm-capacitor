## 🔧 Understanding GGUF Models

GGUF stands for a file format designed to store language models in a compact way. This makes it easier for apps to read. It supports models that are smaller in size but still work well for language tasks.

dust-llm-capacitor runs these GGUF models directly on mobile devices. This means your data stays on your phone and the AI works offline.
